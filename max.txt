Для установки максимального числа активных соединений на сервере nginx, нужно отредактировать конфигурационный файл nginx.conf.

1. Откройте файл nginx.conf с помощью текстового редактора:

sudo nano /etc/nginx/nginx.conf

2. Найдите строку, содержащую директиву worker_connections, которая устанавливает максимальное число активных соединений:

worker_connections 1024;

3. Измените значение на желаемое число соединений (в данном случае - 100):

worker_connections 100;

4. Сохраните изменения, нажав Ctrl + O, а затем выйдите из текстового редактора, нажав Ctrl + X.

5. Перезапустите nginx, чтобы применить изменения:

sudo systemctl restart nginx

Теперь на вашем сервере nginx установлено максимальное число активных соединений, равное 100.
