1. Откройте файл конфигурации Nginx с помощью любого текстового редактора. Обычно он находится в /etc/nginx/nginx.conf.

2. Найдите блок http и добавьте следующие строки внутри него:

error_log /root/log/nginx_error_log error;

3. Сохраните изменения и перезапустите Nginx:

sudo systemctl restart nginx

Теперь все ошибки Nginx будут записываться в файл /root/log/nginx_error_log. Если каталог /root/log не существует, его нужно создать заранее:

sudo mkdir /root/log

Заметьте, что использование каталога /root для хранения логов не является лучшей практикой с точки зрения безопасности. Лучше создать отдельного пользователя и группу для логов и дать им права на запись в соответствующий каталог.
